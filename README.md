# Diff Reader
- python based with Terminal UI
- handles 2 files at a time
- very much intended to NOT be an editor environment, rather a single use, lightweight helper.
- all controls are relevent to what is in "focus", tab to switch focus from A to B.
- load or paste files into their respective inputs and compare in two different modes:
    1) "diff mode" - identifies regional changes and presents with limited surrounding context)
    2) "merge mode" - shows the full common text along with unique elements from each file
- there is an error that may show a blank screen after Export > [Enter].  Hit Enter again to confirm, Escape to cancel.  Under some unknown circumstances, it just doesn't work.  This is a bug being worked out.
